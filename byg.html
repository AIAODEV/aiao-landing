Hvad kan jeg bygge?

Du beskriver din idé til Claude Code i VS Code — og får en rigtig web-app med database, backend, frontend og AI, live på dit eget pocN.aiao.dev. Ingen DevOps, ingen opsætning. Bare idéen.

De fire byggesten

Hver ny POC kommer med det hele klar fra start. Du bygger oven på det ved at tale med Claude.

🖥️

Frontend

En rigtig hjemmeside med AO's eget udseende — klar fra dag ét.

  • Next.js + TypeScript
  • AO-designsystem: farver, typografi, 81 AO-ikoner
  • Valgfri adgangskode-beskyttelse
  • Henter data direkte fra databasen
⚙️

Backend

Et API til logik, beregninger og integration til interne systemer.

  • FastAPI (Python)
  • Egne API-endpoints
  • Nøgler & hemmeligheder sikkert server-side
  • Taler med database + eksterne tjenester
🗄️

Database

Din egen database — opret tabeller ved at bede Claude om det.

  • Supabase (Postgres), dit eget aflukke
  • "opret en tabel til kunder" → klaret
  • Ingen DB-adgang eller SQL-bøvl
  • Sikker adgang pr. række (RLS)
🤖

AI / sprogmodel indbygget

Tilføj intelligens — chat, opsummering, klassifikation, udtræk.

  • Claude- og OpenAI-modeller via ét endpoint
  • Ingen API-nøgle at skaffe
  • Læs, skriv, sortér og svar

…og det her sker helt automatisk 🚀

Du skal aldrig røre servere, hosting eller certifikater. Platformen klarer maskinrummet.

Sådan bygger du — i praksis

Du behøver ikke være udvikler. Du fører en samtale.

Få din POCEn admin opretter den i platformen — du får et repo og et live-link med det samme.
Åbn i Claude Code (VS Code)Beskriv idéen i almindeligt sprog: "byg en side hvor man kan søge i vores produkter og få et AI-svar."
Claude byggerDen skriver frontend, backend og opretter de tabeller der skal bruges. Se det live med make dev.
Del detNår det virker, er det allerede online på pocN.aiao.dev — klar til at vise frem.

Til inspiration

Eksempler fra hverdagen hos AO. Start småt: byg det simpleste der beviser idéen.

🔎 AI-produktsøgningSøg i sortimentet i naturligt sprog og få ét samlet svar — ikke bare en liste.
📦 Leverings-overblikEt dashboard der samler status på ordrer/leveringer ét sted.
💬 Intern FAQ-botEn chat der svarer kolleger ud fra jeres egne dokumenter og vejledninger.
📝 Formular → databaseSaml indmeldinger (fejl, ønsker, tilmeldinger) i en tabel med pænt overblik.
🏷️ Auto-klassifikationLad AI sortere/tagge indkommende tekst (mails, sager, varer).
📊 Lille dashboardVisualisér tal fra en tabel — KPI'er, kampagner, lager, aktivitet.
📄 Dokument-opsummeringIndsæt en lang tekst, få et kort brugbart resumé på dansk.
🧪 Konfigurator / beregnerEn guide der regner sig frem til det rigtige produkt eller pris.
